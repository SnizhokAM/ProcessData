What is ProcessData?
******************

ProcessData is a free Telegram bot for accounting and data organization. Powered by Python 3, its use is governed by the GPL License.


How to install:
***************

1. Install the Python interpreter version 3.6 or higher, if it has not been installed before.
2. Configure dependencies: install or update the python-telegram-bot and telegram packages.
3. Run the main.py script.
4. You can use the report*.py and report.pyw scripts to generate reports.

Що таке ProcessData?
******************

ProcessData - це безкоштовний Telegram бот для обліку та впорядкування даних. Працює в середовищі Python 3, його використання регулюється Ліцензією GPL.


Як встановити:
***************

1. Встановити інтерпретатор мови Python версії 3.6 або вище, якщо його не було встановлено раніше.
2. Налаштувати залежності: встановити або оновити пакети python-telegram-bot та telegram.
3. Запустити скрипт main.py.
4. Для формування звітності можна використати скрипти report*.py та report.pyw.
